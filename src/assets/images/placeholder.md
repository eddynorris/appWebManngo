# Imágenes de la Landing Page

Esta carpeta contiene las imágenes utilizadas en la landing page con NgOptimizedImage.

## Imágenes requeridas:

- `carbon-premium-xl.jpg` (300x200px) - Imagen del carbón premium XL
- `carbon-natural-clasico.jpg` (300x200px) - Imagen del carbón natural clásico  
- `briquetas-artesanales.jpg` (300x200px) - Imagen de las briquetas artesanales

## Características de las imágenes:

- **Formato:** JPG optimizado para web
- **Dimensiones:** 300x200 píxeles (aspecto 3:2)
- **Tamaño:** Máximo 50KB por imagen
- **Calidad:** Optimizada para web (85% calidad)

## Beneficios de NgOptimizedImage:

✅ **Carga diferida automática** - Las imágenes se cargan solo cuando son visibles  
✅ **Formatos modernos** - Soporte automático para WebP/AVIF cuando están disponibles  
✅ **Responsive loading** - Carga de diferentes tamaños según el dispositivo  
✅ **Priority hints** - Marcas de prioridad para imágenes importantes  
✅ **Preconnect automático** - Conecta automáticamente a CDNs de imágenes  

Para reemplazar estas imágenes placeholder, simplemente coloca los archivos reales con estos nombres en esta carpeta.
